## Параллелизация
Реализована с использованием OpenMP путем распараллеливания внешних циклов обработки по строкам изображения в функциях `rotateClockwise`, `rotateCounterClockwise` и `applyGaussianFilter`. Включение параллельной версии осуществляется директивой компиляции `-DUSE_OPENMP` и флагом линковки `-fopenmp` (цель `run_par`), последовательная версия собирается без этих флагов (цель `run_seq`).
